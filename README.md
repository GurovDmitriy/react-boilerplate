# React Boilerplate

---

## npm clean install a project

```bash
npm ci
```

## npm install a package

```bash
npm install
```

## npm run a security audit

```bash
npm audit [fix]
```

## npm check for outdated packages

```bash
npm outdate [--all]
```

## npm update packages

```bash
npm update
```
